# Ad Cuts — Batch 1 (the hook test)

Five finished 9:16 verticals (1080×1920, captions burned in), built from
real roster shoes and the campaign footage. Same body idea, five
different hooks — this batch exists to find out which hook the
algorithm feeds, not to be precious about any one of them.

## The files

| File | Hook | Length | Sound |
| --- | --- | --- | --- |
| ad1-forced-pick.mp4 | Forced pick: Afro Samurai vs Treacherous Waters, live score | 12s | silent — add trending audio in-app |
| ad2-stock-price.mp4 | "This sneaker has a stock price" → real market board | 10s | silent — add trending audio in-app |
| ad3-the-record.mp4 | "3–0. Nobody's beaten it yet." → live battle board | 10s | silent — add trending audio in-app |
| ad4-artist-callout.mp4 | Artist recruit: flipped for triple, artist got $0 → royalty | 13s | scene audio baked in |
| ad5-wrong-answers.mp4 | "Wrong answers only: what's this worth?" | 7s | silent — add trending audio in-app |

Silent on purpose: adding a trending sound inside the TikTok/IG/FB
composer boosts distribution and can't be done if music is baked in.

## The playbook

1. Post all five organic over ~10 days (don't dump same-day; give each
   its own shot). Battle Blast in the admin panel fires the same
   battles to X/Bluesky/Telegram/Discord/Reddit.
2. Watch exactly two numbers per post: 3-second holds (retention) and
   comment count. Ignore likes.
3. Put ALL paid budget behind the single winner. Kill the rest without
   sentiment.
4. The link goes in bio / first comment — "vote in the comments" in the
   caption is what earns reach; the site link converts the curious.
5. When a winner emerges, this kit regenerates endlessly: every new
   battle on the site is a fresh ad1, every big bid is a fresh ad2,
   every win streak is a fresh ad3.

## How they're made (repeatable)

Frames are rendered from HTML using the site's own fonts/tokens
(Anton + Archivo, ink/volt/heat) via headless Chromium, assembled with
ffmpeg. No paid generation involved — new variants cost nothing but
minutes. Ask Claude for "another ad1 with <battle>" and it's a re-run.
